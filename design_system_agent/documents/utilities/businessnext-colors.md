## Usage in Custom CSS

```css
/* Text color */
.my-element {
  color: var(--bnds-g-color-blue-50);
}

/* Background color */
.my-card {
  background-color: var(--bnds-g-color-gray-10);
  border: 1px solid var(--bnds-g-color-gray-30);
}

/* Semi-transparent overlay */
.overlay {
  background-color: var(--bnds-g-color-black-50);
}

/* Hover state */
.button:hover {
  background-color: var(--bnds-g-color-blue-60);
}
```

## Color Naming Convention

Colors follow a 10-100 scale:
- **10-30**: Light tints, backgrounds
- **40-60**: Primary/interactive colors
- **70-100**: Dark shades, text colors

## Semantic Usage

### Status Colors
- **Success**: Use green-50 (`#5eba71`)
- **Error/Danger**: Use red-50 (`#EA001E`)
- **Warning**: Use yellow-30 (`#FCC003`)
- **Info**: Use blue-50 (`#0176D3`)

### UI Elements
- **Primary Action**: blue-50
- **Borders**: gray-30
- **Backgrounds**: gray-10 (white), gray-20 (off-white)
- **Text**: gray-100 (dark), gray-70 (medium)
- **Disabled**: gray-40
